<h2>Was ist HanDeDict?</h2>

<p>HanDeDict ist ein Open Source chinesisch-deutsches Wörterbuch, das kollaborativ erstellt wurde und unter Create Commons (<a href="http://creativecommons.org/licenses/by-sa/2.0/de/" target="_blank">CC BY-SA 2.0 DE</a>) lizenziert ist. Es wurde ursprünglich als eine Übersetzung des CEDICT chinesisch-englischen Wörterbuch konzipiert, und wurde zwischen 2006 und 2011 mit der Unterstützung der Chinesisch-Deutschen Gesellschaft e.V. Hamburg aktiv entwickelt. Im August 2015 wurde die Webseite von HanDeDict unnerreichbar; die <a href="https://web.archive.org/web/20150325174145/http:/www.handedict.de/chinesisch_deutsch.php?mode=about" target="_blank">letzte Aufnahme</a> auf der Wayback Machine vom Internet Archive ist vom 1. August 2015. Die Textdatei mit dem Inhalt vom letzten Stand des Wörterbuchs kann <a href="https://github.com/zydeo/HanDeDict/" target="_blank">von Github heruntergeladen werden</a>. Die Webseite, die Sie jetzt durchsuchen, basiert auf dieser Textdatei.</p>

<h2>Was ist Zydeo?</h2>

<p>In seiner ursprünglichen Form ist <a href="https://zydeo.net/" target="_blank">Zydeo</a> ein Open Source chinesisch-deutsches Wörterbuchprogramm für Windows, das auf den Daten von <a href="http://cc-cedict.org/wiki/" target="_blank">CC-CEDICT</a> basiert. Ich habe es zu meinem eigenen Vergnügen erstellt, als ein Hobbyprojekt, das mir erlaubte, mit einer minimalistischen Benutzeroberfläche zu experimentieren. Ab und zu veröffentliche ich Einträge über dieses Projekt auf dem <a href="https://blog.zydeo.net/" target="_blank">Zydeo-Blog</a>. Der <a href="https://github.com/zydeo/Zydeo" target="_blank">Quellcode</a> ist auf Github zu finden.</p>

<p>Ich hatte es nicht vor, aus demselben Grund eine Online-Version zu entwickeln; es gibt ja zahllose Webseiten im Internet, wo man CEDICT durchsuchen kann. Ich habe aber wohl vor, bald ein ähnliches Projekt wie HanDeDict in die Wege zu leiten: eine Übersetzung von CEDICT in eine andere Sprache, offen für kollaborative Ausbreitung und Weiterentwicklung. Als ich erfuhr, dass HanDeDict aus dem Internet verschwunden ist, wurde mir klar, dass ich Teile von Zydeo relativ leicht wiederverwenden kann, um diese Webseite zu starten und HanDeDict wieder ins Internet zu stellen. Das Ergebnis ist HanDeDict @ Zydeo.</p>

<h2>Zukunftspläne</h2>

<p>In ihrer jetzigen, einfachen Form ermöglicht diese Webseite, HanDeDict durchzusuchen.  In den nächsten Monaten plane ich kleinere Updates mit Verbesserungen. Besonders das Komponent für Handschrifterkennung, das auf einer Portierung von Jordan Kiangs <a href="http://www.kiang.org/jordan/software/hanzilookup/" target="_blank">HanziLookup</a> durch <a href="http://jabtunes.com/notation/chinesestroke.html" target="_blank">Zz85</a> basiert, verdient viel mehr Aufmerksamkeit.
</p>

<p>Letztendlich soll die Seite eine Redaktionsoberfläche erhalten, durch die alle Benutzer vorhandene Einträge verbessern und neue Einträge erstellen können. Ich hoffe, dadurch eine Gruppe aktiver Benutzer zu finden, die HanDeDict wiederbeleben und es weiterentwickeln werden von dem an sich sehr hohen Stand, den das ursprüngliche HanDeDict-Team bis 2011 erreicht hat.</p>
